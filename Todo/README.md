# TODO Code sample

TODO app running using [Firebase!](http://firebase.com) and [ReactJs!](https://facebook.github.io/react/)


### installation

To install `npm` and `bower` dependencies

```
 npm install
 bower install
 
```


### Test 
Run unit tests 


``` 

 gulp test

```


### Run
Start the app locally 
```
 gulp

```
 
 

### Build
Build dist of the app for prodution
```
 gulp --prod

```
